# RiwiFlow - Kanban Task Manager

A modular Single Page Application (SPA) designed to manage software development workflows natively utilizing a Kanban interface layout.

## Features Built

- **SPA Custom Routing**: Seamless views switches without page reloads.
- **Strict Role Enforcement**: Separate operational spaces for `admin` and `coder` profiles.
- **Persistent Storage**: CRUD mutations simulated through a `json-server` REST API.
